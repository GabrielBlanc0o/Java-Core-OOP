## Primero contacto con Spring  - Julio 1

*Que es spring web?*

Esto es lo que permite crear endpoints REST, por asi decirlo cuando alguien haga una peticion HTTP a tu API, esto lo recibe y responde.

*Que es Spring Data JPA ?*

Es la capa que conecta mi codigo Java con la Base de datos, en vez de hacer querys manuales de SQL, escribo clases Java y el las convierte en tablas automaticamente.

*Significado de PostgreSQL Driver en el initializr*

Gracias a este conector podemos conectar Spring y la base de datos PostgreSQL especificamente, sin esto no sabrian como comunicarse entre ellos
