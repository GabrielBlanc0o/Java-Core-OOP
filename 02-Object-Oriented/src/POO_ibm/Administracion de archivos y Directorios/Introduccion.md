# Java File Handling

## Que nos permite manejar los archivos en Java

* Primero esto nos permite hacer lectura y escritura de archivos
* Para interactuar con archivos usamos el paquete java.io 
* Ayuda a la persistencia de datos aunque el programa termine
* Compartir archivos procesados entre diferentes programas
